# SimpleAgenticAIImplementation
Agentic AI that covers the basic use cases of Agents and how to properly approach implementing them

Learn and Be Curious section on Agentic AI, building each individual simple agent pattern listed here: 
https://docs.aws.amazon.com/prescriptive-guidance/latest/agentic-ai-patterns/basic-reasoning-agents.html
 
* Agent patterns
  * Basic reasoning agents
  * Tool-based agents for calling functions
  * Tool-based agents for servers
  * Computer-use agents
  * Coding agents
  * Speech and voice agents
  * Workflow orchestration agents
  * Memory-augmented agents
  * Simulation and test-bed agents
  * Observer and monitoring agents
  * Multi-agent collaboration
